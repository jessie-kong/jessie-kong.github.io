Projects

Wordle “Next Best Guess” Solver
May 2022

As an avid Wordle player, I decided to write a program in Java to assist someone in the online word-guessing game. My program allows a user to enter their guess and the game’s response in tile colors (which provide hints about letters) to eliminate possible words. The algorithm then determines the “next best guess,” or word most likely to reduce the word bank by the greatest amount.

Repo Link: https://github.com/jessie-kong/wordle-solver



Ohio State Robot Project

For my Fundamentals of Engineering Honors course, my team was tasked with constructing and programming a robot to autonomously complete diner tasks: flip burgers, slide tickets, deposit trays, etc. We designed the robot to navigate the course using bump switches, light sensors, and a remote positioning system (though we also learned line-following along the way).

Video Link: https://youtu.be/2m2ahNZw99c



Tetris in C++
December 2021

My classmate and I recreated Tetris using C++ object-oriented programming, modular code, and easily modifiable graphics. We stayed honest to the key features of Pajitnov's original work but added a personal twist aesthetically. The game featured a leaderboard saving high scores, a “next” queue showing upcoming blocks, and a “hold” box allowing the player to save a piece for later.

Walkthrough Link: https://cpb-us-w2.wpmucdn.com/u.osu.edu/dist/c/112906/files/2021/12/Tetris_Walkthrough_Music.mp4
